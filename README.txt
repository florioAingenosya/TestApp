
------------- Test usage -----------------------------------

App Backend : 
	- SQL : gère les données
	- Spring Boot Framework
		- mvn spring-boot:run pour demarrer le serveur
	

App Front :  
	- React JS 
		- npm install
		- npm start 

Fonctionnalités :
	- Liste Enregistrement à l'acceuil
	- Pour se connecter clique sur MON COMPTE puis connexion
		- Login : admin@mail.com
		- Mdp : admin
	- Pour enregister un nouvel utilisateur
		- Clique sur le bouton "Connexion ou inscription avec google"
	- Clique sur l'une des enregistrement pour pouvoir commenter
